Просмотры и ER (Ограничения Telegram API)

⚠️ Внимание: Стандартный Telegram Bot API не имеет методов для получения количества просмотров сообщений в каналах. Для реализации реальных дашбордов с пересылками (Forwards), ER (Engagement Rate) и просмотрами необходимо авторизовать клиентское приложение (Userbot) через MTProto, например с помощью Pyrogram или Telethon, работающее параллельно с этим ботом. То, что вы видите ниже - это базовая статистика работы нейросети в системе.
